# Mbaay Mi

Application agricole locale pour connecter les cultivateurs, partager des annonces et des conseils agricoles.

## Description

Mbaay Mi est une plateforme web dédiée à l'agriculture locale au Sénégal. Elle permet aux agriculteurs de publier des annonces, de partager des conseils et de se connecter avec d'autres professionnels du secteur.

## Fonctionnalités

- **Annonces agricoles** : Publication et consultation d'annonces pour semences, outils, etc.
- **Tableau de bord** : Suivi des abonnements et activités
- **Contact** : Formulaire de contact pour les partenariats
- **Carte interactive** : Localisation des services

## Technologies utilisées

- HTML5
- CSS3
- JavaScript

## Installation

1. Clonez le repository :
   ```bash
   git clone https://github.com/elhadjicheikhouomarfaye-ctrl/mbaay-mi.git
   ```

2. Ouvrez `index.html` dans votre navigateur.
